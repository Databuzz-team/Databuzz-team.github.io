    Hexo Scaffolds란?

    hexo new -[page/post/draft 등] 을 실행했을 때 Default로 들어갈 정보를 입력하는 공간이다.

    1. Scaffolds/post.md 파일 수정하기

    2. New post 추가하기

    3. Title

    4. Tags

    5. Deploy(중요함!!)

    마지막에 Git pull해라는 오류가 뜨면, Pull을 실행하고 다시 sh post.sh로 재업로드해줄것!
